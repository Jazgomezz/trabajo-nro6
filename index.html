3. Las variables, mas conocidas como identificadores, se ajustan a ciertas reglas. Un identificador de JavaScript comienza con una letra, un guión bajo ( _ ) o un signo de dólar ( $ ).
En JavaScript hay ocho tipos de datos: número, BigInt, booleano, string o cadena, valor nulo, indefinido, símbolo y objeto.
